controls to play the game:

W,S,A,D        =  for movement of the player
Toggle Crouch  =  C
Toggle Crouch  =  Z
Space bar      =  jump
Pick up gun    =  F   (can pick up 2 guns)
Equip weapon   =  Q (primary weapon)  , E (secondary weapon)
Fire           =  left click(mouse)
Reload weapon  =  R
